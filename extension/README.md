# Focus Tint Chrome Extension

Focus Tint is a minimal Chrome extension that adds a dim overlay to the active tab so you can focus on what matters.

## Load the extension locally

1. Open `chrome://extensions` in Chrome.
2. Enable **Developer mode** (top-right).
3. Click **Load unpacked**.
4. Select the `extension` folder from this repo.

## Usage

1. Click the **Focus Tint** extension icon.
2. Press **Toggle overlay** to switch the dimming overlay on or off.
3. The popup shows the current state of the overlay.

## Notes

- The overlay will not run on Chrome Web Store pages or internal Chrome URLs.
